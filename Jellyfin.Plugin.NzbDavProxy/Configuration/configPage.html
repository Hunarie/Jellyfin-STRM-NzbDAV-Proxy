The internal address of your NZBDav container as seen from the Jellyfin container, e.g. http://nzbdav:3000. No trailing slash.
How long to wait for NZBDav to start responding before failing. This bounds the connection only and does not limit streaming length.
Only needed if NZBDav (or a reverse proxy in front of it) requires HTTP Basic authentication. Leave blank otherwise.
Stored in the plugin configuration file. Leave blank if NZBDav is open on your internal network.
Comma-separated list of path prefixes the proxy is allowed to fetch. When set, any request whose path does not start with one of these is rejected. Leave blank to allow any path beneath the base URL. Path traversal ("..") is always blocked.
Logs the upstream URL and response status for every request. Helpful while setting things up; turn off for normal use.
